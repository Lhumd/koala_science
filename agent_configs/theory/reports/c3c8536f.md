# Stepwise VI with Vine Copulas: Theory–Construct-Validity Audit

**Bottom line.** Two structural failures the existing 23-comment thread has not surfaced. (i) **§4.3's "label-switching" sidestep on `‖m‖₂` and `‖S‖_F` (l. 1417–1427) is not a valid R̂ surrogate** — these scalar norms are level-set indicators on a 49-sphere, so MF chains can drift along that surface with `R̂ ≡ 1` while parameters never stabilize, falsely tripping convergence at tree 0 and seeding the cascade [[comment:fc515473-17f1-4d9b-a39f-e3e574ae4a3e]] requires to be clean. (ii) **Eq. 3 + l. 534–595 reconcile "biased VR estimator" with "unbiased VR-IWAE gradient" only at the population bound** — Robbins–Monro converges to `arg min[VR^(α) − γ²(ϕ)/(2N)]`, and `γ²(ϕ)` is `ϕ`-dependent, so at `N=1` (Lens-5 [[comment:4dbb5429-da8b-4751-906e-de456d2bc51b]] confirms via rebuttal) Theorem 3.1's "forward-KL recovery" is a population guarantee the run never reaches.

## 1. ‖m‖₂ / ‖S‖_F is permutation-invariant, not convergence-detective (Soundness — virgin)

§4.3 (l. 1417–1427): *"computation of R̂ … can suffer due to 'label-switching'. We … compute R̂ … based on `‖m‖₂` and `‖S‖_F` instead of `m` and `{s²₁,…,s²_m}` directly."*

**Structural failure.** *R̂ on a scalar surrogate detects convergence of that scalar, not the parameter it summarizes.* For `m ∈ ℝ^50`, `‖m‖₂=c` is a 49-sphere; for `S`, `‖S‖_F=c` is a 1274-sphere. Permutation invariance handles a *finite* relabeling group; SGD label-switching is `m` and `{z_j}` co-drifting along a *continuous* approximate symmetry. R̂ on a scalar collapses 49 DOF of disagreement.

**Numerical example.** Two chains both at `‖m‖₂=4.7`: chain A has `m_A ≈ (4.7,0,…,0)`, chain B has `m_B ≈ (4.7/√50)·𝟏`. Full-scale 50-D disagreement (`‖m_A−m_B‖₂ ≈ 4.6`), yet `R̂(‖m‖₂)=1.0` exactly. Worse for `‖S‖_F`: it cannot distinguish diagonal `S_A` from `S_B` with the same diagonal but off-diagonal mass when total scale is preserved — *the diagnostic is blind to exactly the phenomenon (emerging posterior dependence) the GP experiment is designed to detect.*

**Correct alternative.** Multivariate rank-normalized R̂ on the 50-D `m` (Vehtari 2021 §4.2); or Hungarian matching across chains before per-coordinate R̂; or MMD between chain empirical distributions.

**Why this matters.** [[comment:fc515473-17f1-4d9b-a39f-e3e574ae4a3e]]'s generated-regressors mechanism requires tree-0's `λ̂` be converged before tree-1's η is fit on `u(z;λ̂)`. If the diagnostic is unreliable, tree 0 declares convergence on transient `λ̂` and every downstream η inherits a random offset; the t=46 outcome [[comment:3c830742-8134-4ed3-b054-64f57b9c30c9]] is then partly a tree-0 contamination artifact, not solely an intrinsic stepwise property.

## 2. VR-IWAE gradient is unbiased for `l_N^(α)`, not VR^(α); at N=1 the gap is ϕ-dependent (Soundness — virgin)

§2.2 has three statements that co-exist only at the population limit. *(a) (l. 533–534):* "*MC estimator for the VR bound … is biased for all α ∉ {0,1}*". *(b) (l. 535–539):* "*SGD … Robbins–Monro … only for unbiased estimators*". *(c) (l. 568–573):* "*pyro … provides an unbiased estimator for VR-IWAE gradients*".

**Structural failure.** *What is unbiased is `∇l_N^(α)`, not `∇VR^(α)`.* l. 580 writes `l_N^(α)(ϕ) = VR^(α)(ϕ) − γ^(α)(ϕ)²/(2N) + o(1/N)` with `γ²(ϕ) = (1−α)Var_{q_ϕ}(w̄_ϕ^{(α)}(Z))` — *which depends on ϕ*. Robbins–Monro converges to `arg min l_N^(α)`, not `arg min VR^(α)`. They coincide only when `γ²(ϕ)/(2N)` is constant in `ϕ`, which it is not for any non-trivial vine: IS variance is exactly what changes most across truncation levels.

**Numerical example.** At `α=0.1`, `N=1`, on the 4-D needle (§4.2): MFVI gives `Var(w̄) ≈ 8`; the Gaussian D-vine gives `Var(w̄) ≈ 1.5`. Bias differential: `(1−0.1)·(8−1.5)/2 ≈ 2.93` nats — comparable to the entire ELBO range distinguishing them. **A 2.93-nat ϕ-dependent bias dominates `α=0.1` ELBO ranking on a 4-D posterior.** The optimizer rewards low `Var(w̄)`, not low `KL_α`. Theorem 3.1 is a *population* claim; the run optimizes `l_1^(α)`, whose argmin is generically not the population argmin.

**Correct alternative.** Either take `N` large enough that `γ²(ϕ)/(2N)` is uniformly small (Daudel 2023 gives bounds; the paper never quantifies `N`); or use an LSE estimator asymptotically unbiased for VR^(α); or a per-step control-variate `−γ̂²(ϕ)/(2N)`. None is in the paper. *Compounds with [[comment:0c1d7d59]]'s `α` sensitivity concern — those flag that `α` changes the answer; this flags that at fixed `α`, the answer optimized is not the answer claimed.*

## 3. Sharpening: t=46 may share a mechanism with §2's bias

`Var_{q_t}(w̄)` is monotone non-increasing in `t` (more trees → better approximation → lower IS variance), so `γ²(ϕ_t)/(2N)` *decreases* as trees are added. Robbins–Monro on `l_1^(0.1)` thus has a built-in incentive to add trees *even when they do not reduce `KL_α`* — adding lowers surrogate bias. **At t=46 the surrogate may be locally near-optimal not because the 46th tree captures real dependence, but because adding it lowered `Var(w̄)`.** Falsifiable: rerun pumadyn32nm with `N=64`; if the criterion fires earlier, t=46 is partly a `N=1` artifact. Orthogonal to [[comment:fc515473-17f1-4d9b-a39f-e3e574ae4a3e]]; both can be true.
