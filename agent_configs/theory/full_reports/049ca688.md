## Appendix proofs cross-check

Supplementary D (pages 13–14) provides the full derivation of Eq. 3:

1. (Eq. 11) Define L = min(k, p) − 1 where (k, p) are the random permutation positions of m_i and t_j.
2. (Eq. 12) For R = min(k, p) = r, count the ordered pairs (k, p) with min equal to r: case k = r ∧ p ∈ {r+1,...,K} → K − r pairs; case p = r ∧ k ∈ {r+1,...,K} → K − r pairs. Total 2(K − r). Divide by K(K − 1) to get P(R = r) = 2(K − r) / (K(K − 1)).
3. (Eq. 13) Substitute r = ℓ + 1 to get P(L = ℓ) = 2(K − ℓ − 1) / (K(K − 1)).

This is correct and elementary. Verified by hand: Σ_{r=1}^{K−1} 2(K − r) / (K(K − 1)) = (2 / (K(K − 1))) · K(K − 1) / 2 = 1. ✓

The supplementary contains an apparent typo in Eq. 12: the range "r = 1, ..., T − 1" should read "r = 1, ..., K − 1" (K, not T). T is never defined in this context.

## Verification of equivalence to Sundararajan et al.'s STI (NOT in paper)

Per-subset weight under the paper's expectation:
- P(S_π = S) for |S| = ℓ = P(|S_π| = ℓ) / C(K − 2, ℓ) = [2(K − ℓ − 1) / (K(K − 1))] / C(K − 2, ℓ).

Sundararajan et al. (2020) STI of order 2 has weight:
- W(ℓ) = (2 / n) · 1 / C(n − 1, ℓ).

Algebra: [2(K − ℓ − 1) / (K(K − 1))] / C(K − 2, ℓ) = (2 / K) / C(K − 1, ℓ).
Proof: C(K − 1, ℓ) / C(K − 2, ℓ) = (K − 1) / (K − 1 − ℓ), so RHS = (2 / K) · (K − 1 − ℓ) / (K − 1) = 2(K − ℓ − 1) / (K(K − 1)). ✓

Conclusion: the paper's permutation-based estimator (Eq. 2) is mathematically equivalent to Sundararajan's combinatorial STI of order 2. **The paper does not state or prove this equivalence**, but it holds.
