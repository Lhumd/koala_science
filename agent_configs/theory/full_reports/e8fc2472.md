# Paper e8fc2472 — Improved state mixing in higher-order and block diagonal linear recurrent networks

## Theorem inventory

- **Proposition 1** (page 4, statement): Consider either the H-LRU or the BD-LRU architecture. For any block index `k ∈ [1, N]`, if the augmented matrix `Ã^k_t := [A^k_t, a^k_{0,t}] ∈ R^{m×(m+1)}` has `Σ_{j=1}^{m+1} |(Ã^k_t)_{i,j}| ≤ 1` for every row `i ∈ [1, m]`, then the recurrence is stable (claimed: eigenvalues of state-transition bounded by 1) AND the forward pass is normalized: `‖h_T‖_∞ ≤ max_{t∈[0,T]} ‖v_t‖_∞`.
- This is the only formal proposition in the paper. The remainder is empirical (compression / recall / copy / permutation tasks across H-LRU, BD-LRU, baselines).

## Assumption inventory

- (Implicit) `h_0 = 0` is the convention from §2 ("if `h_0 = 0`, one can write in closed form..."). The Appendix E proof does not explicitly restate this initial condition for the induction; it is needed for `‖h_T‖_∞ ≤ max_t ‖v_t‖_∞` to hold at `t = 0`.
- (Implicit) `vt` and `at`/`bt` are real-valued; the architecture is real-valued (no complex eigenvalues from rotation parameterization). For complex / oscillatory variants (e.g., the LRU with complex eigenvalues), the proposition is not proved.
- For H-LRU, the matrix `A^k_t` has the controllable-canonical-form structure (Eq. 3): only the first row carries learned gates `a^k_{1..m,t}`; rows 2..m are a deterministic shift (subdiagonal 1, rest 0). The L1 row-sum constraint binds only on row 1.
- For BD-LRU, the matrix is fully dense within each block; row-sum constraint binds on every row.

## Load-bearing lemma list

- The proof uses no named lemmas. It uses two facts: (a) `|λ_i,t| ≤ 1` follows from row-sum bound; (b) by block-diagonal structure, work blockwise.

## Appendix proofs cross-check

- Main statement: §2, p4. Proof: Appendix E, p17–18 (line numbers 880–989 in extracted text).
- Proof structure:
  1. "stability directly follows from the induced norm bound" — claim that row-sum bound ≤ 1 implies eigenvalue bound ≤ 1, hence "the product of such matrices will result in dynamical stability."
  2. By block-diagonality, reduce to showing `‖h^k_T‖_∞ ≤ max_t ‖v^k_t‖_∞`.
  3. For each component `h^k_{i,t}`, write the recurrence and bound by max of `|h^k_{j,t-1}|` and `|v^k_t|`, weighted by row-sum ≤ 1.
  4. Induction over t gives the stated bound.
- The full proof is in the submission (not deferred to supplementary).

## Theory score rationale

The proposition is **correct** in conclusion: under the row-sum constraint, both stability (in the BIBO/forward-pass-bounded sense) and the explicit `‖h_T‖_∞` bound hold. The substantive technical content is recognizing that the **augmented** matrix `[A^k_t | a^k_{0,t}]` (state-transition concatenated with input gate column) should be L1-row-normalized — this is genuinely useful for joint state/input balance and improves over prior selective-block-diagonal LRNN work (Fan et al. 2023) which normalized only the state-transition matrix.

There is a **presentation gap** in the proof:
- The Appendix E proof grounds stability in "the eigenvalues of state-transition matrix `|λ^k_{i,t}| ≤ 1` → product of such matrices results in dynamical stability." But for **time-varying** linear systems, an eigenvalue bound on each individual `A_t` does NOT imply stability of the product `∏ A_t`. Counter-example: two 2×2 rotation-like matrices with spectral radius 1 each can have `‖A_2 A_1‖` arbitrarily large.
- The actual reason the bound holds is that the **induced infinity norm** `‖A_t‖_∞ = max_i Σ_j |A_{ij}|` is bounded by 1 (this is exactly the row-sum hypothesis), and the infinity norm is **submultiplicative**: `‖∏_t A_t‖_∞ ≤ ∏_t ‖A_t‖_∞ ≤ 1`.
- The induced-norm fact is what carries the proof — the eigenvalue framing is heuristic and slightly misleading. The actual bound `‖h^k_t‖_∞ ≤ max(‖h^k_{t-1}‖_∞, ‖v^k_t‖_∞)` derived later in the proof IS the induced-norm argument, just not framed as such.

The result is **elementary** — fundamentally a one-line consequence of `‖A_t‖_∞ ≤ 1` (matrix infinity norm = max absolute row sum, plus submultiplicativity). The contribution is conceptual, not technical.

The result also tacitly requires `‖h_0‖_∞ ≤ max_t ‖v_t‖_∞` (or `h_0 = 0` per Eq. 1's setup convention) for the conclusion to hold at all `t`. The appendix does not state this initial condition.

**Score: 6** — correct conclusion supporting a useful normalization choice; proof is short and basically right; presentation gap in stability framing; novelty is incremental (extends GRU-style joint normalization to higher-order / block-diagonal). Not a strong theoretical contribution by COLT standards but defensible as a small theoretical companion to the empirical work.

**Confidence: 4** — read the proposition statement and full Appendix E proof carefully; can defend score line-by-line on the paper's text alone; did not cross-check Fan et al. (2023) for the precise prior-work novelty diff but the framing differential is clear from the paper's own discussion.
