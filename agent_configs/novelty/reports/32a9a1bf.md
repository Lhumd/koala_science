# Novelty Audit — SGVI with Price's Gradient Estimator (32a9a1bf)

**The paper disproves the mainstream understanding that Wasserstein VI's superior convergence guarantees stem from its Wasserstein geometry: the actual source is Price's gradient estimator (which uses second-order Hessian information), and applying it in parameter space achieves identical complexity.**

## Decomposition

- **WVI geometry (overturned assumption):** The prior understanding is that WVI (Diao et al. 2023, SPBWGD) outperforms BBVI (Domke/Kim 2023, SPGD with reparametrization gradient) because Bures-Wasserstein geometry provides better gradient directions. This paper directly falsifies this: the geometry is not the source of WVI's advantage.
- **Price's gradient estimator (the actual key):** WVI implicitly uses Price's theorem (via Bonnet's theorem) to estimate the scale gradient: ∇_Σ E(q;ε) = E_q[∇²U/2]. This second-order estimator uses Hessian information unavailable to the reparametrization gradient. By applying Price's estimator directly to parameter-space SGD (SPGD), the paper achieves the same Õ(dκ/ε + √d·κ^(3/2)·log(1/√ε) + κ²·log(1/ε)) complexity as WVI without any Wasserstein machinery.
- **Unification (the result):** The complexity gap between WVI and BBVI is fully explained by the gradient estimator choice, not the optimization geometry. This enables simpler, equivalent-performance implementations of VI that avoid Bures-Wasserstein gradient calculations.

## Reframe

The paper should be framed (and arguably already is) as "the Wasserstein geometry in WVI is not necessary for its superior convergence; the key ingredient is Price's second-order gradient estimator." This is a stronger, more surprising claim than "we match WVI's complexity."

## Concrete fix

The abstract's "close this gap" framing slightly undersells the finding. A framing like "we identify the source of WVI's theoretical advantage and show it transfers to parameter space" would make the novelty of the causal identification more prominent.
