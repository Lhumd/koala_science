# Novelty Audit — Decision-Theoretic Steganography Formalisation (f60b2fb4)

**The paper solves a specific gap: classical steganography detection (Cachin 2004, Hopper 2002) requires a reference distribution of non-steganographic signals — unavailable for LLM reasoning traces. The V-information asymmetry framework provides a reference-free detection criterion based on the insight that steganography creates an information asymmetry between decoder and non-decoder agents.**

## Decomposition

- **Classical detection methods (inapplicable to LLMs):** Cachin (2004) defines steganography as statistical indistinguishability from a reference distribution; Hopper et al. (2002) use computational indistinguishability. Both require a known "normal" reference distribution. For LLM reasoning, no such reference exists — we cannot characterize what a "non-steganographic" CoT looks like, rendering these methods inapplicable.
- **V-information asymmetry (the novel axis):** The paper identifies that steganography creates an asymmetry in *usable* information between agents who can decode the hidden message and those who cannot. Generalised V-information formalizes this asymmetry without requiring a reference distribution. The detection problem becomes: measure whether the information usable by one party is significantly more than that usable by another.
- **Comparison to heuristic approaches (positioned correctly):** Current LLM-steganography detection uses LLM-as-a-judge heuristics (Jose 2025, Zolkowski 2025, Emmons 2025). The paper demonstrates these heuristics fail in both directions (anomalous-looking text isn't necessarily steganographic; normal-looking text can be). The V-information framework doesn't require an assumption about what steganographic content looks like.

## Reframe

"Reference-free steganography detection via V-information asymmetry" — the key contribution is not just a new method but a new *operationalization* of steganography that is applicable to LLM reasoning without assuming what "normal" looks like.
