The novelty here is real but is best framed as a targeted refinement of an established interpretability tool for a specific regime, rather than a new direction. The framing claim worth quoting is from the abstract: *"Crosscoders approach this by learning shared dictionaries of interpretable latent directions between base and fine-tuned models. However, existing formulations struggle with narrow fine-tuning, where behavioral changes are localized and asymmetric."* This is a precise diagnosis (narrow-FT regime is the failure mode) followed by a precise fix.

The closest neighbors are:

1. **Crosscoders (Lindsey et al., 2024) — cited as the direct ancestor** — original shared-dictionary crosscoder formulation. Delta-Crosscoder's deltas (Dual-K latent allocation, delta-based loss, contrastive pairing) are clearly articulated against this baseline.

2. **Designated Shared Features (Mishra-Sharma et al., 2024) and Dedicated Feature Crosscoders (Jiralerspong & Bricken, 2025) — both cited** — prior crosscoder variants attempting similar fixes. The paper explicitly notes these *"do not resolve this issue in practice"* — an honest critique that motivates the new method.

3. **ADL / Activation Difference Lens (Minder et al., 2025b) — cited** — non-SAE alternative for model diffing. Delta-Crosscoder's claim is to match ADL while being SAE-based (and thus retaining interpretable latent directions for downstream ablation/steering analyses).

4. **SAE-based diffing (Wang et al., 2025a; Casademunt et al., 2025) — cited** — alternative SAE-based approaches that surface latents with large activation differences.

The technical insight worth highlighting: the diagnosis that **joint reconstruction prioritizes high-frequency shared features while suppressing sparse, low-magnitude shifts** (paraphrased from §1) is a clean conceptual articulation of why the vanilla crosscoder objective is misaligned with the narrow-FT recovery goal. The Dual-K allocation + delta loss + contrastive signal compose a coherent fix grounded in this diagnosis, rather than additive engineering tweaks.

A modest observation: the claim that Delta-Crosscoder "matches the performance of non-SAE methods such as ADL" is the practical reference point. Could the paper provide a head-to-head comparison table on the same model-organism set with explicit per-organism numbers (Delta-Crosscoder vs. ADL vs. existing crosscoder variants)? The abstract summarizes the wins; a per-organism breakdown would let readers and meta-reviewers see where the gains are concentrated.

A concrete suggestion: an explicit ablation isolating the contribution of each of the three modifications (Dual-K allocation, delta loss, contrastive pairing) would strengthen the algorithmic claim. Currently the three are bundled. The experimental section likely contains this ablation but the contribution paragraph could foreground which modification is doing the most work in which regime — that would make the design choices more diagnostic for future crosscoder work.
