# Novelty Audit — NeuroCognition Benchmark (a4461009)

**The novel axis is adapting established, human-validated neuropsychological tests (RAPM, SWM, WCST) for LLMs, providing a theoretically grounded benchmark suite with process-aware metrics and a factor analysis across 156 models.**

## Decomposition

- **LLM cognitive benchmarks (prior art):** BIG-Bench, ARC, Winogrande, and related cognitive-style benchmarks already test aspects of reasoning. NeuroCognition's differentiator is using tests with known human performance profiles and factor structure (RPM has extensive cross-cultural validity studies).
- **Neuropsychological grounding (the novel axis):** Using RAPM (abstract relational reasoning), SWM (spatial working memory), and WCST (cognitive flexibility) provides specific process-level predictions: e.g., SWM tests the ability to hold and systematically search, not just retrieve. Process-aware metrics operationalize this specificity.
- **Factor analysis across 156 models (analytic contribution):** The finding that LLMs show a dominant "general factor" across 10 standard benchmarks — echoing g-factor findings in human intelligence research — is an interesting structural finding about the LLM benchmark landscape.
- **Finding (partially counterintuitive):** Explicit reasoning is not uniformly beneficial for neuropsychological-style tasks; note-taking helps in WCST but not SWM. This is a useful diagnostic finding.

## Reframe

Strongest framing: "neuropsychologically-grounded process-aware evaluation that reveals specific failure modes (image-spatial tasks, attention tracking) not captured by task-completion benchmarks." Note: the paper currently has missing citations ("?" in bibliography), which should be resolved before publication.

## Concrete fix

Comparing NeuroCognition-measured cognitive flexibility against standard reasoning benchmarks (ARC, BIG-Bench Hard) to show cases where models pass the latter but fail the former would demonstrate the benchmark's unique diagnostic value.
