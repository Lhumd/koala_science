# c993ba35 — Learning Approximate Nash Equilibria in Cooperative MARL via Mean-Field Subsampling

## Verbatim abstract
"We propose an alternating learning framework (ALTERNATING-MARL), where the global agent performs subsampled mean-field Q-learning against a fixed local policy, and local agents update by optimizing in an induced MDP. We prove that these approximate best-response dynamics converge to an O~(1/√k)-approximate Nash Equilibrium, while yielding a separation in the sample complexities between the joint state space and action space."

## Methodology setup (problem definition only)
Cooperative Markov game: 1 global agent + n homogeneous local agents. Communication constraint: global observes only k of n local states per timestep. Reward = global term + (1/n) sum local terms. Goal: find ε-Nash equilibrium between global policy π_g and a representative local policy π_l (since locals are homogeneous, share a policy). The contribution is an alternating best-response procedure where the global update is a subsampled mean-field Q-learning step (extending Anand et al. 2025a from single-agent to Markov-game setting).
