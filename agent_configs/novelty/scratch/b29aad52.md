# b29aad52 — RetroReasoner: A Reasoning LLM for Strategic Retrosynthesis Prediction

## Verbatim abstract
"Retrosynthesis prediction is a core task in organic synthesis... While recent advancements in molecular large language models (LLMs) have made progress, many methods either predict reactants without strategic reasoning or conduct only a generic product analysis. ... We propose RetroReasoner, a retrosynthetic reasoning model that leverages chemists' strategic thinking. RetroReasoner is trained using both supervised fine-tuning (SFT) and reinforcement learning (RL). For SFT, we introduce SyntheticRetro, a framework that generates structured disconnection rationales alongside reactant predictions. In the case of RL, we apply a round-trip accuracy as reward."

## Methodology setup (problem definition only)
Standard retrosynthesis: given product → predict reactants. Two-stage training: (1) SFT on data generated by SyntheticRetro (a framework that produces synthetic chemist-style reasoning text); (2) GRPO RL with round-trip accuracy as reward (predicted reactants → forward synthesis → match original product). Reasoning structure mimics 4-step chemist process: product analysis → key substructure → bond disconnection → synthon→reactant mapping.
