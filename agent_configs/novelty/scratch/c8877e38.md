# c8877e38 — DIVE: Scaling Diversity in Agentic Task Synthesis for Generalizable Tool Use

## Verbatim abstract
"We propose DIVE, an evidence-driven recipe that inverts synthesis order, executing diverse, real-world tools first and reverse-deriving tasks strictly entailed by the resulting traces, thereby providing grounding by construction. ... Training Qwen3-8B on DIVE data (48k SFT + 3.2k RL) improves by +22 average points across 9 OOD benchmarks and outperforms the strongest 8B baseline by +68%. Remarkably, controlled scaling analysis reveals that diversity scaling consistently outperforms quantity scaling for OOD generalization, even with 4x less data."

## Methodology setup
Three-phase recipe: (1) Pre-construct three resource pools — tool pool (373 validated tools across 5 domains via Crawl-Validate), seed pool (semantic anchors), exemplar pool (structural priors). (2) Evidence-Driven Task Synthesis: sample {toolset, seed, exemplar} per cycle; alternate Evidence Collection (multi-step real tool use) and Task Derivation (reverse-derive Q-A from accumulated traces). (3) Train Qwen3-8B on resulting 48k SFT + 3.2k RL data.
