## 1. Selective read

**Abstract.** "3DGSNav, a novel ZSON framework that embeds 3D Gaussian Splatting (3DGS) as persistent memory for VLMs to enhance spatial reasoning. Through active perception, 3DGSNav incrementally constructs a 3DGS representation ... enabling trajectory-guided free-viewpoint rendering of frontier-aware first-person views ... structured visual prompts and integrate them with Chain-of-Thought (CoT) prompting to further improve VLM reasoning."

## 4b. Closest prior

1. **VLFM (Yokoyama et al. 2024).** Closest VLM-ZSON neighbor. Delta: VLFM uses 2D similarity value maps; 3DGSNav uses 3DGS persistent memory + free-viewpoint rendering for re-verification. Substantive — different memory representation.
2. **BeliefMapNav (Zhou et al. 2025, concurrent).** Closest 3D-memory ZSON neighbor with voxel-based map. Delta: 3DGS vs voxel grid; 3DGSNav adds free-viewpoint *novel-view* rendering, voxel grids don't.
3. **ApexNav (Zhang et al. 2025, concurrent).** Closest neighbor on rule-based + VLM ZSON. Different memory primitive.
4. **Zhang 2023 (3D semantic point clouds for navigation).** Closer to 3DGSNav on memory primitive (continuous 3D); 3DGSNav adds VLM integration + active perception.
5. **NeRF-based scene memory work (precursor; not directly cited but a clear lineage).** 3DGS is the speed/quality successor to NeRF for scene memory.

## 4c. Pseudo-novelty patterns

- Rebrand? **No.**
- Domain transfer? **Yes — substantive.** 3DGS (a rendering primitive originally for novel-view synthesis) → persistent memory for VLM-guided navigation. The new task does need active acquisition, viewpoint selection, frontier-aware rendering — all of which the paper addresses.
- Composition without insight? **Yes, partial.** 3DGS + active perception + structured visual prompts + CoT is a 4-component composition. The free-viewpoint *re-verification* primitive (use 3DGS to render new viewpoints during navigation for target confirmation) is the load-bearing original element. The other components are largely stitching.
- Resolution bump? No.
