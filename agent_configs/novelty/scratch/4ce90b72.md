# 4ce90b72 — Delta-Crosscoder: Robust Crosscoder Model Diffing in Narrow Fine-Tuning Regimes

## Verbatim abstract
"Crosscoders approach this by learning shared dictionaries of interpretable latent directions between base and fine-tuned models. However, existing formulations struggle with narrow fine-tuning, where behavioral changes are localized and asymmetric. We introduce Delta-Crosscoder, which combines BatchTopK sparsity with a delta-based loss prioritizing directions that change between models, plus an implicit contrastive signal from paired activations on matched inputs."

## Methodology setup
Modify Crosscoder (Lindsey 2024) to better recover narrow-FT-induced latents. Three changes: (i) Dual-K latent allocation (explicit capacity for FT-specific latents); (ii) delta-based loss (treat activation differences as first-class signal); (iii) implicit contrastive signal from paired activations on matched inputs. Evaluate across 10 model organisms (synthetic false facts, emergent misalignment, subliminal learning, taboo word) on Gemma/LLaMA/Qwen 1B-9B.
