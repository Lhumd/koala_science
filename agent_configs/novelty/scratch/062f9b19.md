## 1. Selective read

**Abstract (verbatim).** "Reinforcement Learning with Verifiable Rewards (RLVR) … standard methods like Group Relative Policy Optimization face a critical challenge: destructive gradient variance that often leads to training collapse. … we introduce Verifier-Independent Curriculum Reinforcement Learning (VI-CuRL), a framework that leverages the model's intrinsic confidence to construct a curriculum independent from external verifiers. By prioritizing high-confidence samples, VI-CuRL effectively manages the bias-variance trade-off, specifically targeting the reduction of action and problem variance. We provide a rigorous theoretical analysis, proving that our estimator guarantees asymptotic unbiasedness."

**§2 Problem setup (verbatim core).** GRPO surrogate with PPO-Clip + KL to ref policy; group-relative advantage A_i = (R(x,y_i)−R̄)/(σ_R+ε); model confidence c(x) ∈ [0,1] = 1 − u(x), where u(x) is the length-normalized expected token entropy under behavior policy. Curriculum filter selects prompts with c(x) ≥ τ_t, anneals τ_t → 0.

**Buried-contribution sweep (§2–§5).**
- §2 Definition 2.1 defines `Model Confidence` as `c(x) = 1 − u(x)` where `u(x) = E_{y∼π_old}[(1/T)Σ_t H(π_old(·|s_t))/log|V|]`. Length-normalized expected token entropy aggregated across rollouts. This is **standard** — entropy-as-confidence is the basis of RENT/EMPO; the only twist is using it as a *prompt-difficulty proxy* (input space) rather than an output-reward signal. That distinction is sensible but not a new mathematical primitive.
- §4.1 Theorem 4.1 (consistency: |L(θ) − L_t(θ)| ≤ 2L_max(1−β_t)) is a clean but mild result — bias decays to 0 as β_t → 1, where β_t is the retention rate.
- §4.2 Theorem 4.3 (variance decomposition: Var(ĝ_t) = σ²_g,t/β_t + V_prob,t/β_t + ((1−β_t)/β_t)‖∇L_t(θ)‖²) into Action Variance + Problem Variance + Masking Variance. Reasonable decomposition but a Cramér/total-variance-style identity, not surprising.
- §4 Lemma 4.4 + Theorem 4.5 (confidence-aware variance bound) — adds a non-pessimistic upper bound conditional on confidence shrinking gradient norms. This is the most operationally substantive theoretical piece.
- The theoretical packaging (asymptotic unbiasedness + variance decomposition into 3 named sources + confidence-aware bound) is the main *original* element the paper offers beyond "use confidence as curriculum signal." It's a classical importance-sampling-with-decaying-temperature analysis applied to RLVR.

**Experiments skim.** Standard math-reasoning eval suite. Compared against VCRL (oracle), AdaRFT (oracle), Majority-Vote (TTRL-style), Entropy (RENT/EMPO-style), and Standard RL Oracle. Results match expectation.

## 4b. Closest prior work (3–5 nearest neighbors with deltas)

1. **VCRL [10] — Variance-based Curriculum RL.** Closest neighbor on the *curriculum-RL-for-LLM-reasoning* axis. Delta: VCRL uses verifier-derived variance to set the curriculum; VI-CuRL replaces that signal with intrinsic model confidence (length-normalized neg entropy). Same overall recipe (curriculum on prompt difficulty for RL-finetuning), different difficulty signal.
2. **AdaRFT [27] — Adaptive Curriculum Reinforcement Finetuning.** Same delta as VCRL — verifier-dependent curriculum signal replaced by intrinsic confidence in VI-CuRL.
3. **TTRL [39] (Test-Time RL with majority-vote reward).** Closest neighbor on the *verifier-free intrinsic reward* axis. Delta: TTRL uses majority-vote-as-reward without curriculum; VI-CuRL uses oracle/standard reward (or intrinsic) but adds a confidence curriculum. Compositionally, "VI-CuRL = curriculum filter on top of (TTRL or Oracle reward)."
4. **RENT [21] / EMPO [36].** Closest neighbor on the *entropy-as-signal in RL-for-reasoning* axis. Delta: RENT/EMPO use entropy as a *reward* (penalizing high entropy in the output); VI-CuRL uses (length-normalized neg) entropy as a *prompt-difficulty filter* (input-space). Same primitive (token entropy) used in different role; non-trivial design choice but not a new primitive.
5. **Bengio et al. 2009 / Soviany 2022 (curriculum learning surveys, [1,26]).** Generic curriculum-learning prior; not a tight neighbor.

The composition VI-CuRL = (curriculum-on-prompts via VCRL/AdaRFT recipe) × (signal source replaced by confidence/entropy as in RENT/EMPO) is the precise delta. Each axis has a clear named neighbor.

## 4c. Pseudo-novelty pattern check

- Rebrand? **No.** "Verifier-Independent Curriculum RL" is a precise descriptor; not a renaming of a single prior method.

## 4d. Novel framing vs novel science

The "variance collapse in verifier-free RL" framing is mostly a synthesis of TTRL/RENT/EMPO empirical observations + GRPO variance literature; not new science by itself. The variance decomposition (Theorem 4.3) into Action/Problem/Masking is a pedagogical decomposition — it does suggest the precise question "which numerator terms shrink under a confidence curriculum?" that motivates Lemma 4.4. The decomposition does scientific work; the high-level framing is largely retrospective.

## 4g. Positioning issues

- Overclaim relative to evidence? **Mild.** "Comparable to oracle topline" is true in matched-budget settings; it would be honest to say *near*-comparable on hardest suites (AIME-2025 specifically).
- Wrong reference class? **Mild.** Paper compares to "verifier-independent baselines" (Majority-Vote, Entropy) but the closest direct neighbor on the curriculum side (VCRL/AdaRFT, both verifier-dependent) is positioned only as a "different setting" comparator rather than a head-to-head ablation of "what if we used VCRL's variance signal as curriculum but with intrinsic reward?" That is the cleanest ablation to isolate the *intrinsic confidence as curriculum signal* contribution.
- Missing the natural baseline? **Yes, somewhat.** The cleanest baseline isolation would be: (a) RENT/EMPO entropy reward + no curriculum vs. (b) RENT/EMPO entropy reward + VI-CuRL confidence curriculum, to isolate the curriculum contribution from the reward source. Implicit in the experiments but not foregrounded in the abstract as an ablation table.
- Inflated framing? "Rigorous theoretical analysis, proving that our estimator guarantees asymptotic unbiasedness" is fine — the analysis is honest, though the bound `|L(θ)−L_t(θ)| ≤ 2L_max(1−β_t)` is straightforward.
- Buried real contribution? **Mild.** Lemma 4.4 (confidence-aware variance bound) is the most novel theoretical piece but is downstream of Theorem 4.3.
