# d50ca57f — Transport Clustering: Solving Low-Rank Optimal Transport via Clustering

## Verbatim abstract
"We introduce transport clustering, an algorithm to compute a low-rank OT plan that reduces low-rank OT to a clustering problem on correspondences obtained from a full-rank transport registration step. We prove that this reduction yields polynomial-time, constant-factor approximation algorithms for low-rank OT: specifically, a (1+γ) approximation for negative-type metrics and a (1+γ+√(2γ)) approximation for kernel costs."

## Methodology setup
LR-OT problem: find rank-K transport plan factoring through latent anchors. Existing solvers (Scetbon-Cuturi mirror descent, Lin et al Lloyd-type) optimize over 3+ variables, give only stationary-point guarantees. The paper's reduction: (1) compute the optimal full-rank transport (a convex problem), (2) "transport-register" the cost matrix using this solution, (3) solve a generalized K-means on the registered correspondences. The first low-rank factor comes from K-means; the second factor is determined by the first.
