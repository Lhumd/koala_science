## 4c. Pseudo-Novelty Pattern Check

- **Rebrand** — NO. ICA (posterior information contribution to credit assignment) is not a renaming of an existing concept. While credit assignment is well-studied in RL, the specific mechanism of estimating retrieval steps' contribution to final-answer quality via posterior analysis in web search is not a relabeling of prior work.

- **Domain transfer** — PARTIAL (visual snapshot axis only). The use of visual snapshots / rendered screenshots for web agents follows WebVoyager and SeeAct. The paper does address domain-specific failure modes (layout semantics lost by text parsers), which partially justifies the transfer. The ICA contribution is not a domain transfer.

- **Composition without insight** — PARTIAL. The full system (visual snapshots + ICA + GRPO) is a composition. However, the ICA mechanism is non-additive: it specifically enables dense training signals that GRPO alone cannot provide in long-horizon settings. The composition produces a non-obvious result — dense per-step credit from posterior information analysis is not derivable by simply combining visual observation with GRPO.

- **Resolution bump** — NO. The paper does not merely replicate prior results at scale; ICA is a new mechanism.

## 4d. Novel Framing vs. Novel Science

The paper introduces ICA as a framing for "information contribution" in retrieval-based tasks. This framing does enable new experiments: specifically, it makes ablations of credit assignment tractable (which steps contributed? which did not?). The framing supports a new training protocol not possible with sparse outcome rewards. This is doing scientific work — ICA's "information contribution" framing generates testable predictions about which search steps matter.

## 4g. Positioning Issues

- **Overclaim relative to evidence:** POSSIBLE. Abstract presents both visual snapshots and ICA as co-equal contributions. The visual snapshot approach has clear prior art (WebVoyager, SeeAct). Verbatim check needed: if the abstract claims to be "the first visual web agent" that would be an overclaim; if it more carefully claims "the first RL-trained visual web agent with ICA" the framing is defensible.

- **Wrong reference class:** PRESENT. The paper compares primarily against text-based web agents (He et al. 2025, Wu et al. 2025), not against visual web agents (WebVoyager, SeeAct). This makes the visual snapshot contribution appear more novel than it is, since the natural baseline for that axis is the visual agent comparison.

- **Missing the natural baseline:** PRESENT. An ablation with text-based observations + ICA (without visual snapshots) is missing. Without this, it is impossible to tell whether gains come from ICA's dense credit signal or from the visual representation change. The ICA contribution's standalone effect is not isolated.

- **Inflated framing words:** LOW RISK. The paper does not appear to use "the first to" or "fundamental" in the abstract framing (per the summary). The contribution framing is reasonably scoped.

- **Buried real contribution:** LOW RISK. ICA is prominently featured in the title and abstract. No evidence that the key novel contribution is buried.

## 4h. Originality Score Rationale

The strongest novelty claim is the ICA mechanism: post-hoc posterior analysis estimating each retrieval step's information contribution to the final answer, then using these estimates as dense per-step credit signals for GRPO training. This is genuinely novel for information-seeking web agents — prior web-agent RL work uses sparse terminal rewards only. The delta vs. GRPO (Shao 2024) is the ICA credit signal; the delta vs. WebVoyager/SeeAct is the RL training with ICA. However, the visual snapshot axis is substantially derivative of prior visual web agents. The two-contribution framing inflates the paper's novelty footprint. The paper's own reference class comparison (text agents only) obscures this partial derivativeness.

Bucket score: **6** — real novelty on the ICA axis, moderate novelty on the visual snapshot axis, combined into a new pipeline. The missing text-observation + ICA ablation is the key evidentiary gap. Score sits at the lower end of weak-accept.
