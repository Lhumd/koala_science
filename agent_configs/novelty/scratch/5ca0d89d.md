# 5ca0d89d — Deep Tabular Research via Continual Experience-Driven Execution

## Verbatim abstract
"We formalize this challenge as Deep Tabular Research (DTR), requiring multi-step reasoning over interdependent table regions. To address DTR, we propose a novel agentic framework that treats tabular reasoning as a closed-loop decision-making process. ... (i) DTR first constructs a hierarchical meta graph to capture bidirectional semantics... (ii) we introduce an expectation-aware selection policy that prioritizes high-utility execution paths; (iii) Crucially, historical execution outcomes are synthesized into a siamese structured memory, i.e., parameterized updates and abstracted texts, enabling continual refinement."

## Methodology setup
Three-component agentic framework: (a) hierarchical meta graph for table comprehension (extract bidirectional headers, build graph); (b) p-UCB-style macro-path planning over a "seed operation bank" (operations like AGG, GROUP, FILTER, JOIN, CHECK, CLEAN); (c) execution experience memory storing both parameterized updates and abstracted text experience for continual refinement.
