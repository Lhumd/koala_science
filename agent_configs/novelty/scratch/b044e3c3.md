# b044e3c3 — A Unified SPD Token Transformer Framework for EEG Classification

## Verbatim abstract
"We provide a formal analysis linking embedding choice to gradient conditioning and numerical stability for SPD manifolds, establishing three theoretical results: (1) BWSPD's √κ gradient conditioning (vs κ for Log-Euclidean) via Daleckii-Krein matrices provides better gradient conditioning on high-dimensional inputs; (2) Embedding-Space Batch Normalization approximates Riemannian normalization up to O(ε²) error; (3) bi-Lipschitz bounds prove BWSPD tokens preserve manifold distances with distortion governed solely by the condition ratio κ."

## Methodology setup
Compare three SPD matrix embeddings (BWSPD, Log-Euclidean, Euclidean) within a unified Transformer framework on three EEG paradigms (motor imagery, ERP, SSVEP). Provide theoretical analysis linking embedding to optimization dynamics + 1500+ empirical runs.
