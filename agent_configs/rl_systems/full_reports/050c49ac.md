## Load-bearing claim inventory
1. "AffordanceGrasp-R1 consistently outperforms SOTA methods on benchmark datasets" (abstract) → Table 1: SOTA on HANDAL/GraspNet/3DOI
2. "real-world robotic grasping evaluations further validate its robustness and generalization" (abstract) → Table 3: 80%/72% vs AffordanceNet 62%/50%

## Construct inventory
**Claim**: GRPO RL "significantly enhances reasoning capability and semantic alignment"
- **Construct claimed**: reasoning capability enhancement from RL
- **Operational metric**: gIoU/cIoU on HANDAL† and 3DOI (reasoning subsets)
- **Aspect NOT measured**: whether the reasoning improvement is specifically from RL's exploration mechanism vs just additional training compute on the same data distribution (second SFT stage achieves similar numbers)

## Reward / objective inventory
RL reward components (Stage 2):
1. Format reward: thinking tags + answer format compliance
2. Non-repeat reward: penalizes repeated reasoning patterns
3. Accuracy reward: bounding box IoU > 0.5, bounding box L1 distance < 10px, keypoint L1 distance < 30px

Comparison: The SFT second stage (CoT-SFT+SFT row, Table 4) uses exactly the same training data and task. The RL reward provides no new information beyond what the CoT supervision already provides, except via exploration.

## Counter-hypothesis list
1. **Second-SFT equivalence**: CoT-SFT+SFT = 65.2/64.2 (Main gIoU/cIoU) vs CoT-SFT+GRPO = 65.5/65.0. The difference is 0.3/0.8 points. RL adds near-zero value over a second SFT stage with same data. NOT ruled out.
2. **SAM2-LoRA dominance**: The final system adds SAM2-LoRA, which takes CoT-SFT+GRPO (64.5/65.9 All) to 66.9/67.1 All — a +2.4/1.2 gain from SAM2-LoRA alone. SAM2-LoRA is not compared against a non-GRPO baseline (CoT-SFT+SFT+SAM2-LoRA). NOT ruled out.

## Framing-vs-demonstration check
- "synergistically combines SFT and RL...significantly enhancing reasoning capability" vs Table 4: CoT-SFT+GRPO Reasoning: 65.1/64.1 vs CoT-SFT+SFT Reasoning: 64.5/64.3 → GRPO adds +0.6/−0.2 on reasoning. "Significantly" is not supported by ablation.
- "robustness and generalization under complex language-conditioned manipulation scenarios" — tested on 10 objects, one robot arm, one baseline (AffordanceNet). Limited diversity.

## Mechanism attribution audit
- Paper introduces: (1) CoT cold-start SFT, (2) GRPO RL, (3) SAM2-LoRA. All three isolated in Table 4.
- Table 4 shows GRPO ≈ SFT (within 0.3-0.8 gIoU points). The paper claims GRPO's "exploration" mechanism is key, but the ablation does not demonstrate GRPO >> SFT.
- The missing baseline: CoT-SFT+SFT+SAM2-LoRA — would test whether SAM2-LoRA alone (without RL) achieves the full gain.
- This is a soft score-3 trigger: mechanism attribution is unclear (RL vs SFT second stage), though the ablation is present.

## Headline-number denominator audit
- No denominator switch detected.
- Real-world: 10 trials per task, total ~100 trials per condition. Statistically meaningful for 18% gap.

## Soundness score rationale
Score 4, confidence 4.

- Key weakness: GRPO ≈ SFT second stage (diff < 0.8 gIoU), yet paper attributes gains to RL specifically ("synergistically combines SFT and RL, significantly enhancing reasoning")
- Missing baseline: CoT-SFT+SFT+SAM2-LoRA to isolate SAM2-LoRA contribution independent of RL
- One score-3 trigger (mechanism attribution unclear) with offsetting strength (real-world validation, SOTA on benchmarks)
